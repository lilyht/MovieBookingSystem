# MovieBookingSystem

一个电影订票系统 :movie_camera:

#### 运行环境

- Python 3.7
- Flask
- MySQL

#### 运行方法

建立数据库后，运行app.py，web访问 http://localhost:9292/ （占用9292端口）

```bash
python app.py
```
